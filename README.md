# Sistema de Gestión de Inventarios Mejorado

Este proyecto corresponde al desarrollo de un Sistema de Gestión de Inventarios en Python,
aplicando Programación Orientada a Objetos, almacenamiento en archivos y manejo de excepciones.

El sistema permite:
- Añadir productos
- Eliminar productos
- Actualizar cantidad y precio
- Buscar productos por nombre
- Mostrar el inventario completo
- Guardar y recuperar datos desde un archivo de texto

---

## Estructura del Proyecto

Sistema_Inventario_POO_Mejorado
│
├── inventario.py
├── main.py   ← Archivo principal
├── producto.py
└── inventario.txt (se crea automáticamente)

---

- El archivo `inventario.txt` se crea automáticamente si no existe.
- Los cambios se guardan después de cada operación.
- Al reiniciar el programa, los productos se cargan automáticamente.
